# MPV Docusaurus publication template

Minimal docs-only Docusaurus template used as a reference, bootstrap template and adapter fixture.

Required manifest: `docs-template.json`.

Runtime environment consumed by `docusaurus.config.ts`:

- `DOCS_SITE_URL`
- `DOCS_BASE_PATH`
- `DOCS_TITLE`
- `DOCS_PORTAL_NAME`
- `DOCS_PORTAL_URL`

The publisher action copies neutral Site content into `docs/` and optional static assets into `static/` before running the build.
