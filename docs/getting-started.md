# Primeros pasos

Esta página muestra cómo convertir el scaffold inicial en una documentación propia.

## 1. Revisa la estructura

La página de inicio vive en [`index.md`](./index.md). Las secciones usan carpetas con su propio `index.md`, como [Conceptos](./concepts/index.md) y [Guías](./guides/index.md).

## 2. Ajusta el frontmatter

Usa `title`, `description`, `label`, `order` y `hidden` para probar la navegación neutral y su conversión a motores como Docusaurus o Fumadocs.

:::note
El contenido debe seguir siendo portable: evita importar componentes específicos de un motor salvo que aceptes perder portabilidad.
:::

## 3. Añade tu primer cambio

Continúa con [editar la primera página](./guides/first-edit.md).
