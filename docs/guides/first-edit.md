# Editar la primera página

1. Abre [`index.md`](../index.md).
2. Cambia el título o la descripción.
3. Añade un enlace hacia [Arquitectura](../concepts/architecture.md).
4. Comprueba que la imagen de ejemplo sigue publicándose correctamente.

![Diagrama de ejemplo](/img/sample-diagram.svg)
