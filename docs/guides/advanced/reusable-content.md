# Contenido reutilizable

Esta página vive en una subsección anidada para validar navegación profunda y enlaces relativos.

## Buenas prácticas

- Mantén los ejemplos cerca de la página que los explica.
- Referencia assets estáticos desde `/img/...` cuando deban publicarse como recursos globales.
- Prefiere admonitions portables con `remark-directive`.

:::caution
No dependas de componentes exclusivos de Docusaurus o Fumadocs si necesitas poder cambiar de motor.
:::
